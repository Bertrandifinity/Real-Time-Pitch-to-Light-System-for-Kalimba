# Real-Time Pitch-to-Light System for Kalimba
Real-Time Pitch-to-Light System for Kalimba is a C++ Linux embedded project that captures live kalimba audio, detects musical pitch in real time, and instantly maps each note to synchronized LED colour output, demonstrating low-latency, event-driven embedded system design on Raspberry Pi.
